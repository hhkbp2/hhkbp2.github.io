My Errata of UNPe3

Mar 6th, 2011 | Comments

This is my errata of Unix Network Programming volume 1: The Sockets Networking API, 3 edition

  1. p248 line 4 of paragraph 1:
    “vefiry by running netstat on freebsd)”
    should be
    “verify by running netstat on freebsd4)”
    freebsd and freebsd4 are two different machines, according to the hosts and networks topology for this text.

  2. p324 line 2 of paragraph 2, below the first testga command output, “-h bsdi” should be “-h freebsd4”.

  3. p681 the section title “6.4 TCP Echo Server Using Threads” should be “26.4 TCP Echo Server Using Threads”.
