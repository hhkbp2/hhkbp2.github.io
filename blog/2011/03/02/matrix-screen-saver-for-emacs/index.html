Matrix Screen Saver for Emacs

Mar 2nd, 2011 | Comments 

"What is the Matrix?
Control."
    -- Morpheus

对于广大的程序员来说，骇客帝国应该是一部很有吸引力的电影，因为它既有很丰富的商业大片的元素，也有很多我们熟悉的事物：英雄，导师，爱情，格斗和子弹时间，代码，骇客，人类和机器，现实与虚拟，过去和未来预言等。而它的确是一部成功的电影，无论是从它取得的票房上来看，还是从它在世界范围产生的影响来看。

片中为人津津乐道的地方很多，关于宗教哲学，关于现实，关于信念等等，而引起我思考最多的是the Matrix这个AI，机器，人类社会的未来的某一个发展阶段。

什么是the Matrix呢？Morpheus说，control。控制，与被控制。即使片中那个未来社会目前还无法预见，但是其实无需等待那个AI产生的未来，看看现时当下，就很容易发现the Matrix无处不在: gfw; 高房价，control了多少人的生活; 艰难的决定，control了你用哪个客户端软件…

而当年我们的伟大领导RMS正是因为私有软件的版权对人的控制而开始了自由软件运动。

废话少说，长话短说，回到正题，今天说的是又一个碟形世界般的故事：

历史的潮流不断向前推动，在这个不知从何时开始，又不知到何时结束的宏大时代的某一天的某一个不太清醒的时候，镜头转向阿图因大陆，在亚洲东部某个国家南方某个魔法并不盛行的小镇，有一个年轻人在家喝了点酒，晕晕乎乎之余仿照骇客帝国的场景，为GNU Emacs写一个Matrix屏保…

于是就有了zone-matrix.el。

以Linux作桌面系统也有很多时间了，长时间运行Linux，有个屏保对显示器保护比较好,在X window下有很多的屏保可用，但有时只有一个只支持几种颜色的终端。在Emacs的标准发行包中有一个zone.el，提供了几个比较不错的屏保，zone-matrix.el重用了zone.el的框架。希望给那些长时间在终端Emacs下工作的朋友多一种选择。

下载安装：
到这里下载zone-matrix.el，zone-matrix-settings.el，zone-settings.el几个文件，并放到load-path目录中。
然后在dot emacs文件中加入

(require 'zone-settings)

即可。效果图如下：

终端:

主要在终端下使用，终端下能用的颜色不多，表现力还是很有限的，有时间的话再添加多一些变化。也添加了支持X的部分代码，所以在X下也能用。
